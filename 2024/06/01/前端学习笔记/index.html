HTML

简介

HTML（hyper text markup language）：超文本标记语言，用于创建网页的一种标准语言
超文本：该文本文档内含有超链接
标记语言：通过不同的标记来区分各部分，比如*和|等符号。标记和标签是一个意思

    表单（form）

    ​

    可以理解为网页中可以填写的表
    第一个form后面加一个action=”地址”，这句话的意思是将输入框得到的数据传递给该地址去处理，还可以加一种属性method，但是现在用不上，以后再学

    表单里要设置各种表单元素，不然网页中什么都看不到。

    输入框：，右边加一句name设置名字，这样后台可以通过name区分输入框；加placeholder=”提示语”，可以在框内看到，比如请输入密码
    如果是要输入密码的话，把text改成password，则输入的东西都会显示为***
    提交的按钮：把type改成button即可，后面加一句values=”名字”，给该按钮添加名字。还可以直接把type改成submit，自动变成提交按钮，点下去会跳转
    单选框：type改成radio，在尖括号右边加一句话，为该单选框的名字，比如男，则单选框右边会提示男。其他框右边加字也有这种效果
    复选框：type为checkbox，连续写三行，每行右边加对应的名字，则三个框都能勾选
    多行文本域：不用input type，直接输入texteara，里面可以设置列数column和行数row

    表格（table）

    跟表单的区别在于，表单中的内容是需要用户输入的，而表格中的内容是已经写好给别人看的
    跟表单一样，刚创建完table不会在网页上显示任何东西，需要添加属性行（tr）和表格数据（td）
    在行内（两个尖括号）添加td即可
    边框：在table的右边设置border=”1”，该1的意思为显示边框，不可以为其他数字，右边还可以加一个属性cellspacing，指两个边框之间的距离
    表格宽度：table右边加width，设置的数字为像素
    对齐：align，默认靠左，靠右用right，居中用center
    除了table以外，tr和td都可以用上面这些标签
    列合并（colspan）：这个比较重要，在td右边加colspan=””，数字代表要合并的列数，但是仅限于该行，合并完之后就看不到这几列之间的边框了。列合并实际上是指将一个列的宽度变为两个列的，所以要删除原来在的那个列
    行合并（rowspan）：这个有点奇怪，也是在td上操作，可能是这样操作的范围更细，可以删除两小行格子之间的边框。行合并之后要删掉原来那行
    表的标题：
    表头：第一行的内容一般是概括性内容，在tr里面用td（heading），会自动居中（表格中间）
    表的名字：caption，加在table的声明下面

    布局（division，简称div）

    划分每个部分对应的区域
    元素：标签加上标签内的内容
    块级元素：独占一行的元素，无论该元素内的内容有多少，该元素都占网页一行的宽度（除非自行修改了）。常用块级元素：div，p，p的作用：可以规定哪些东西放在同一段里面，不用换行
    行内元素：没有占一行，其长度和宽度只由本身内容来决定
    span：用来给行内元素添加样0式，可以用来强调，之后会讲

    写网页的步骤：用div划分区域；创建table，form等标签；用p分段；在不同的段内编写内容

    CSS

    cascading style sheets，层叠样式表，cascade：连锁反应，sheet：表格
    html相当于是房子的结构，css就是给房子做装修

    使用方法

    • 内部样式
    • 行内样式
    • 外联样式

    内部样式（选择器）

    基本用法

    在head里面，新建一对style，选中div，设置好样式，然后body里面的div全部会按照上面规定的样式来。如果div外面有p，该p不会按照div的样式来。但如果p在div里面，则p也会按照div的样式

    一对大括号里面的叫做声明块，其中的每一行叫做一条声明，style里面的可以叫做选择器。

    行内样式（style）

    顾名思义，在行内规定样式，即在尖括号里的右侧设置样式（这个好像不是选择器）

    .

    在p的右边设置各种style。这里有一个问题，如果内部样式里已经设置好了p的样式，但是行内设置的样式不一样，此时称为样式层叠，系统只能选择其中一个。style里的优先级比选择器里的更高，所以最终样式会按style里的来

    外联样式

    基本用法 
    <link rel="stylesheet" href="">

    在html文件外部建一个后缀为.css的文件，把要设置的样式语句全都写在css文件里，然后把css文件引入html文件。
    引入方式：直接打link，会弹出来，href里面填入该文件名（可能是绝对路径）
    这样规定样式的好处是，如果要设置的样式特别多，在html文件里规定会显得特别复杂，此时在css文件里设置好会更简洁
    然后就是css文件里的内容怎么写了，因为上面都已经声明了链接是stylesheet类型，所以不用写两个style尖括号了，直接用选择器就好了

写网页的步骤：用div划分区域；创建table，form等标签；用p分段；在不同的段内编写内容 CSScascading style sheets，层叠样式表，cascade：连锁反应，sheet：表格html相当于是房子的结构，css就是给房子做装修 语法1div{color:red;font-size:20px;} 上面是css里的选择器，用一对大括号包裹该div的样式，以上是其中两个样式 使用方法 内部样式 行内样式 外联样式 内部样式（选择器） 基本用法 div{ color: aqua; font-size: 30px; } 在head里面，新建一对style，选中div，设置好样式，然后body里面的div全部会按照上面规定的样式来。如果div外面有p，该p不会按照div的样式来。但如果p在div里面，则p也会按照div的样式 一对大括号里面的叫做声明块，其中的每一行叫做一条声明，style里面的可以叫做选择器。 行内样式（style）顾名思义，在行内规定样式，即在尖括号里的右侧设置样式（这个好像不是选择器） . 在p的右边设置各种style。这里有一个问题，如果内部样式里已经设置好了p的样式，但是行内设置的样式不一样，此时称为样式层叠，系统只能选择其中一个。style里的优先级比选择器里的更高，所以最终样式会按style里的来 外联样式 基本用法 <link rel="stylesheet" href=""> 在html文件外部建一个后缀为.css的文件，把要设置的样式语句全都写在css文件里，然后把css文件引入html文件。引入方式：直接打link，会弹出来，href里面填入该文件名（可能是绝对路径）这样规定样式的好处是，如果要设置的样式特别多，在html文件里规定会显得特别复杂，此时在css文件里设置好会更简洁然后就是css文件里的内容怎么写了，因为上面都已经声明了链接是stylesheet类型，所以不用写两个style尖括号了，直接用选择器就好了
